Cipher Typer

real time encryption decryption and audilization of Morse code

Check it out

I built this website becouse I couldn't find a good looking morese code translator anywhere. You simply type in the textbox and as you type it is translated in to Morse code. The interesting part is that I used the web audio API to allow the user to listen to the morse code they have typed in by generating the sound on the fly using the broswer.

If you are writing in morse code be sure not to use any special charachters or capitals, they are not ddefined in morese and will result in all further output not working
